This project is a smart chatbot application called SmartChat that can work in two different ways based on the user’s needs. In normal mode, it behaves like a regular AI chatbot that can answer general questions and help with everyday queries. In document mode, the chatbot becomes restricted and answers only using the documents uploaded by the user. It does not use outside knowledge or guess answers. If the information is not available in the documents, it clearly says that it does not have enough information. This makes the system more reliable and controlled compared to typical chatbots.

The scope of this project includes building a clean chat interface where users can have multiple conversations, along with a separate space to upload and manage documents. Users can upload files, view them, delete them, and choose which documents the chatbot should use. The chatbot automatically changes its behavior based on this selection. It also shows clear indicators to let the user know which mode is active and where the answers are coming from. The system focuses on clarity, simplicity, and user control rather than complexity.
The main goal of this project is to create a chatbot that gives users full control over the information it uses while avoiding incorrect or made-up answers. It is designed to be useful in real-world situations where accuracy matters, such as reading reports, studying materials, or working with company documents. This project can be used by students, professionals, and anyone who wants a reliable assistant that can answer questions based on specific documents instead of general knowledge.
